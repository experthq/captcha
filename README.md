# ExpertHQ Captcha Reader example
This small app shows the basics of connecting to EXHQ API.

## Getting Started

Install dependencies
```
pnpm install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
